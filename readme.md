# visualFish

nostale fishing automation.
Config is hot-reloaded on save.

## Installation

Using:

- Python 3.9
- OpenCv

Venv (optional, recomended)
```powershell
$ python3 -m venv venv
$ .\venv\Scripts\activate
```

```powershell
$ pip install -r requirements.txt
$ python .\src\main.py
```

Build with (for now)
```powershell
pyinstaller main.spec
```
